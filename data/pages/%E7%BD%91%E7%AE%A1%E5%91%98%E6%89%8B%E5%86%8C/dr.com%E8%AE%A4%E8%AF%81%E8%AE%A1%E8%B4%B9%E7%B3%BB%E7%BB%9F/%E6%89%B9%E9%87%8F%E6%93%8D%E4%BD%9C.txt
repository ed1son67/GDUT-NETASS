=====  批量收费 =====

 **用户管理 > 批量业务 > 批量收费**

{{ :网管员手册:dr.com认证计费系统:批量收费.png |}}

批量收费这个功能比较简单，只需要在模板中设置好用户的学号和金额，然后导入即可。

导入的模板可以点击页面当中蓝色按钮 ''保存模板'' ，也可以 [[http://10.0.3.26:8080/DrcomManager/batch.do?P=batch_createExcelTemplate&type=batch_prepay&effective=1&reopenByBatchCharge=0|点此下载]]

在模板中按照格式添加数据，如下图所示

{{ :网管员手册:dr.com认证计费系统:格式.png |}}

需要注意：

  * 帐号必须准确无误地对应上续费单上的信息
  * 金额精度为分，建议加上小数点后两位，支持负数扣费
  * 如果有特殊情况，可以在备注中说明

导入模板整理好后，在页面的操作步骤第三步中选择刚刚我们整理好的文件

第四步中需要勾选：**记录收费日志** 以及 **将系统停机账号复通**

{{ :网管员手册:dr.com认证计费系统:勾选.png |}}

确认无误后，点击第五步的上传即可完成批量收费操作

===== 批量修改资料 =====

 **用户管理 > 批量业务 > 批量修改资料**

在这个栏目中，我们使用条件查询来筛选我们想要的用户

{{ :网管员手册:dr.com认证计费系统:条件查询.png |}}

如果我们所需要更多的筛选条件，可以通过右边的 ''添加查询条件'' 进行添加

{{ :网管员手册:dr.com认证计费系统:添加查询条件.png |}}

输入完我们的筛选条件后，我们需要设置我们需要导出或者修改的内容，帐号是必选的，在 ''字段选择'' 中选择相应的字段

{{ :网管员手册:dr.com认证计费系统:字段选项.png |}}

选择完后，点击 ''查询''  即可，我们就可以在下面的表格中看到筛选出来的数据

如果需要批量修改数据，则在 ''统一修改'' 这一行填写数据，然后全选结果

{{ :网管员手册:dr.com认证计费系统:统一设置.png |}}

如果只是需要导出数据，则全选结果后点击 ''提交'' 即可

{{ :网管员手册:dr.com认证计费系统:提交.png |}}

提交后，可以在 ''执行结果'' 处查看执行结果，如果是导出的，可以点击那个 xls 文件，然后便会导出刚刚选择的用户及其信息。

==== 实例 ====

=== 批量清绑楼栋 ===

由于系统缺陷，无法查询到具体接入的交换机是哪一台，因此在进行批量清绑的时候，只能清除整个楼栋的绑定。

需要说明的是，一个楼栋统一使用一个 PVLAN ，东一 PVLAN 为 101，西十四 PVLAN 为 134，如此类推

同样添加一个查询条件：绑定 PVLAN

{{ :网管员手册:dr.com认证计费系统:pvlan.png |}}

输入制定楼栋的 PVLAN，然后在 ''字段选项'' 中选择：绑定 PVLAN 以及绑定 CVLAN，然后执行查询

{{ :网管员手册:dr.com认证计费系统:cvlan.png |}}

点击勾选本页所有结果，同时他会提示是否勾选全部，点击他即可

{{ :网管员手册:dr.com认证计费系统:全部勾选.png |}}

为了清绑，我们只需要将 PVLAN 和 CVLAN 改成 0 即可，在统一设置中各自输入 0

{{ :网管员手册:dr.com认证计费系统:0.png |}}

这时候全部结果都会被修改为 0 ，确认无误后点击 ''提交'' 即可
===== 筛选说明 =====

==== 左 / 右匹配 ====

左 / 右匹配，顾名思义就是进行模糊匹配的。

左匹配，意思就是匹配开头的内容，如果我们在学号中输入 ''3116'' 并且使用左匹配，那么系统就会帮我们匹配以 ''3116'' 开头的学号，类似于 ''3116*''

左匹配，意思就是匹配结尾的内容，如果我们在学号中输入 ''0001'' 并且使用右匹配，那么系统就会帮我们匹配以 ''0001'' 结尾的学号，类似于 ''*0001''

==== > = < 匹配 ====

如果这个选项是一个纯数字的文本框，那么我们将有机会使用 > = < 匹配。
